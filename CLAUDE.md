# HCTwebsite

## Project Overview

A brocure website for a professional cleaning company

## Tech Stack

- HTML5
- CSS3 (animations, 3D transforms, custom properties)
- Vanilla JavaScript (ES6+)

## Project Structure

All website files live in the `HCTwebsite/` subdirectory. Always place HTML, CSS, JS, and any other website assets there.

```
HCTwebsite/
  index.html    — Main page
  style.css     — Styling and animations
  script.js     — Interactive behaviour
  Logo.png      — Company logo (used in nav and footer)
```

## Assets

- **Logo:** `HCTwebsite/Logo.png` — The official company logo. Use this image wherever the company logo is displayed (navigation bar, footer, etc.).
